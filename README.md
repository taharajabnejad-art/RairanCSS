# رایران

خب سلام! 👋  
اگه تا اینجا رسیدی یعنی یا خیلی دنبال یه ابزار درست‌وحسابی برای ظاهر سایتتی، یا داری الکی همه ریپوهای گیت‌هاب رو می‌گردی 😄  
در هر دو حالت، خوش اومدی به رایران!

رایران یه ابزار سبک و جمع‌وجوره برای اینکه ظاهر سایتت رو قشنگ، مرتب و تمیز کنی، بدون اینکه درگیر هزار تا دردسر و پیچیدگی بشی.

## 🎨 قالب‌ها (چون ظاهر مهمه، قبول کنیم!)

رایران چند تا ظاهر آماده داره:

- مدرن (برای اینکه بگی “من خفن‌ام” 😎)
- شیشه‌ای (برای حس آینده‌ای و وایب خاص)
- کلاسیک (برای آدم‌های آروم و منطقی)

فقط فایلش رو عوض کن، کل ظاهر عوض میشه. جادو نیست، CSSه 😄

---

## 🚀 ویژگی‌ها (چرا اصلاً باید استفاده کنی؟)

- سبک و سریع (نه مثل بعضی چیزا که سایتو خفه می‌کنن)
- مناسب سایت‌های فارسی
- ظاهر تمیز و قابل قبول (حتی جلوی آدم سخت‌گیر)
- قابل توسعه (یعنی هرچی خواستی خودت اضافه کن، ما ناراحت نمی‌شیم 😄)
