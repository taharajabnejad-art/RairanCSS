# رایران

![Rairan Banner](banner.png)

خب سلام 👋  
اگه اینجا رسیدی یعنی یا دنبال یه ابزار درست‌حسابی برای ظاهر سایتتی، یا داری الکی تو گیت‌هاب می‌چرخی 😄  
در هر دو حالت خوش اومدی به رایران!

رایران یه ابزار سبک برای ساخت ظاهر قشنگ و مرتب سایت‌هاست که کار طراحی رو راحت، سریع و بی‌دردسر می‌کنه.

بعدش می‌تونی از کلاس‌های آماده استفاده کنی:

```html id="rmd001"

  <p>خوش اومدی به دنیای رایران</p>
</div>

  ورود (یا فرار 😄)
</button>
```

## 🎨 قالب‌ها

رایران چند تا ظاهر آماده داره:

مدرن 😎 (برای اینکه حرفه‌ای به نظر بیای)

شیشه‌ای ✨ (برای وایب آینده‌ای)

کلاسیک 🧠 (برای آدم‌های آروم و منطقی)


فقط فایل قالب رو عوض کن، کل ظاهر عوض میشه. بله، همینقدر ساده.
