# رایران

خب سلام! 👋  
اگه تا اینجا رسیدی یعنی یا دنبال یه ابزار درست‌حسابی برای ظاهر سایتی، یا داری الکی تو گیت‌هاب می‌چرخی 😄  
در هر دو حالت خوش اومدی به رایران!

رایران یه ابزار سبک و جمع‌وجوره برای اینکه ظاهر سایتت رو قشنگ، مرتب و تمیز کنی، بدون دردسر و پیچیدگی‌های اضافی.

## 🎨 قالب‌ها (چون ظاهر مهمه، قبول کنیم)

رایران چند تا ظاهر آماده داره:

- مدرن 😎 (برای اینکه حرفه‌ای به نظر بیای)
- شیشه‌ای ✨ (برای وایب آینده‌ای)
- کلاسیک 🧠 (برای آدم‌های منطقی و آروم)

فقط فایل رو عوض کن، کل ظاهر عوض میشه. بله، همینقدر ساده.
